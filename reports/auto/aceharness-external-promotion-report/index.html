Integration

B

工具 / 数据接入

通过 MCP、内部 API、数据库查询连接业务工具。

C

协作渠道接入

通过 IM、Webhook、审批入口触发和回传流程状态。

D

Skill 资产分发

E

产品 API 嵌入
